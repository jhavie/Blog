Typescript

Typescript是什么

TypeScript是一种由微软开发和维护的免费开源编程语言。它是一个强类型的JavaScript超集，可编译为纯JavaScript。它是一种用于应用级JavaScript开发的语言。

Typescript相对于Javascript的优势

  • 支持强类型和静态类型特性
  • 支持面向对象编程概念，类、接口、继承、泛型
  • 支持可选参数特性
  • 编译代码 开发期间可以突出显示错误
  • 支持模块

Typescript的缺点

  • 需要很长时间来编译代码
  • 不支持抽象类
  • 浏览器运行Typescript 需要一个编译步骤将Typescript转换成Javascript
  • 要使用任何第三库，必须使用定义文件，并不是所有第三方库都有可用的定义文件
  • 类型定义文件的质量

Typescript的原始数据类型

  • number
  • string
  • boolean
  • null
  • void（void类型: Void是不返回任何类型值的函数的返回类型。如果没有可用的数据类型，则使用它）
  • undefined
  • unknow
    可以一定程度上保证类型的安全
  • never
    一个函数永远执行不完 或者 循环执行不完 则为never
  • any
  • enum 枚举
    enum Color { red = 1, blue = 2, yellow = 3 }
  • tupple 元组
    let a: [number, string] = [1, '2'] 固定长度 固定类型的数组
  • union 联合
    let uni: number | string 一个变量可以同时支持多个类型
  • Nullable 可空类型
  • Literal 预定义类型
    let type: 0 | 1 | 2 一个变量只能是0或1或2
